# CodeHS Overview

**Role:** Gentle on-ramp to Python logic before local hardware programming.

## Strengths
- Zero setup friction
- Fast iteration & auto-run
- Built-in quizzes reinforce syntax

## Limitations
| Area | Limitation | Result |
|------|------------|--------|
| Libraries | Restricted | Focus on core logic |
| Hardware | None | Need transition to EV3 |
| Files | Single-script mindset | Later learn project structure |

## Exit Criteria
Students can:
- Use loops & conditionals fluently
- Write & call functions with parameters
- Trace simple algorithms

Next step: [PyBricks](pybricks.md)
