# Tooling Roadmap (MVP)

| Phase | Course | Tool | Purpose |
|-------|--------|------|---------|
| 1 | R1 | CodeHS | Intro Python logic (no installs) |
| 2 | R1 | PyBricks (EV3) | Hardware interaction |
| 2 | R1 | Arduino IDE (later) | Microcontroller fundamentals |
| 3 | R1 | Basic Git (VS Code) | Version snapshots |
| Future R2 | R2 | VS Code Hub | Multi-language workspace |
| Future R2 | R2 | ESP32 / Pico | Advanced MCU features |
| Future R2 | R2 | ROS2 | Distributed robotics |
| Future R2 | R2 | CNC / PCB | Fabrication & electronics depth |

## Transition Criteria (R1)
Move from CodeHS to PyBricks when students can: write functions, loops, and conditionals confidently.
