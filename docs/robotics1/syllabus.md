# Robotics 1 Syllabus (Draft MVP)

| Item | Details (Draft) |
|------|-----------------|
| Audience | Grades 9–10 |
| Platforms | LEGO EV3 (PyBricks), Arduino Uno (later), Codrone EDU (later) |
| Languages | Python (CodeHS → PyBricks), C++ (Arduino later) |
| Design | Hand sketching → CAD → 3D print (later units) |

## Description
Foundational robotics: sensing, actuation, movement basics, algorithmic thinking, and introductory design.

## Early Learning Objectives
- Explain what defines a robot.
- Use pseudocode & flowcharts to plan logic.
- Write simple Python functions & loops.
- Control EV3 motors and read basic sensors.

## Assessment (Initial Framework)
| Category | Weight (Draft) |
|----------|----------------|
| Labs & Notebooks | 30% |
| Projects | 35% |
| Quizzes / Checks | 15% |
| Collaboration / Participation | 10% |
| Final Challenge | 10% |

## Safety (Preview)
- Treat hardware gently; no unplugging under power.
- Keep workspace clear of clutter.
- Report damaged components.

## Roadmap Preview
Intro/History → Algorithmic Thinking → Python (CodeHS) → EV3 Hardware → Movement → (Later) Drone & Arduino → Design/CAD/3D Printing.
