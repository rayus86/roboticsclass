# Glossary (MVP)

| Term | Definition |
|------|------------|
| Robot | A system that senses, processes, and acts autonomously or semi-autonomously. |
| Sensor | Component that measures a physical quantity. |
| Actuator | Device that moves or controls a mechanism. |
| Loop | Repetition structure in code. |
| Pseudocode | Structured natural language to describe algorithm steps. |
